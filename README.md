: v0.10.1においてパック2つ目以降はスキルを発動させないとした形で100回分ずつではなく500回分ずつ取ってスコアの良いものを選ぶ。

- __v0.12.2__
: AI起動時に v0.9.0 : v0.10.1 : v0.11.1 = 85 : 14.25 : 0.25 の比の確率でいずれかのAIとして動く。ただし内包するv0.5.0の評価方法においてchain数の評価方法を変えてある。使用する乱数種は現在時間から生成する。
